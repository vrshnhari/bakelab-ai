# BakeLab AI

A polished static preview of BakeLab AI, an AI-inspired baking assistant for recipe generation, pantry baking, dietary-aware substitutions, and guided baking steps.

## Deploy On Vercel

1. Create a new GitHub repository.
2. Upload these files to the repository root:
   - `index.html`
   - `package.json`
   - `vercel.json`
   - `README.md`
3. Commit the files.
4. In Vercel, click **Add New Project**.
5. Import the GitHub repository.
6. Keep the default settings and click **Deploy**.

No build command is required because this is a static site.
